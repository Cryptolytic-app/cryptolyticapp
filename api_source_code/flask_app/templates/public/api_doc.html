API Documentation



Introduction

Welcome to the Cryptolytic API documentation. The API’s are split into two categories: trading and arbitrage, which are both public.


API

The Cryptolytic public API endpints can be accessed at the URL: cryptolyticapp.com


Requests

All requests and responses are json content type and follow typical HTTP response status codes for success and failure.

Success

A successful response is indicated by HTTP status code 200 and may contain an optional body. If the response has a body it will be documented under each resource below.


Errors

Unless otherwise stated, errors to bad requests will respond with HTTP 4xx or status codes.

Status Code Error
400 Bad Request – Invalid request format
401 Unauthorized – Invalid API Key
402 Forbidden – You do not have access to the requested resource
403 Not Found
500 Internal Server Error – We had a problem with our server

Types

Timestamps

Unless otherwise specified, all timestamps from API are returned in PST to the second. Make sure you can parse the following PST format. Most modern languages and libraries will handle this without issues.


Rate Limits

Trading API

We throttle public endpoints by IP: 10 requests per minute. When a rate limit is exceeded, a status of 429 Too Many Requests will be returned.

Arbitrage API

We throttle public endpoints by IP: 30 requests per minute. When a rate limit is exceeded, a status of 429 Too Many Requests will be returned.


Trade API

Information about the latest trade opportunities.

HTTP Request

GET /trade

http://www.cryptolyticapp.com/trade

Field Description
results Dictionary of trade results
exchange Crypto Market Exchange
trading_pair Cryptovurrency Trading Pair
p_time Prediction Start Time (PST)
period Prediction Time Window
prediction Prediction for Price Direction

Arbitrage API

Information about the latest arbitrage opportunities. Unavailable between 7pm - 5am PST

HTTP Request

GET /arbitrage

http://www.cryptolyticapp.com/arbitrage

Field Description
results Dictionary of arbitrage results
exchange_1 First Crypto Market Exchange
exchange_2 Second Crypto Market Exchange
trading_pair Cryptocurrency Trading Pair
p_time Prediction Start Time (PST)
prediction Arbitrage Prediction Opportunity
